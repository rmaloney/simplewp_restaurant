== Description ==

WOW Slider is a jQuery image slider for Wordpress with stunning visual effects (Blast, Fly, Blinds, Squares, 
Slices, Basic, Fade, Ken Burns, Stack, Stack vertical and Basic linear) and tons of professionally made templates. 
WOW Slider is packed with a point-and-click wizard to create fantastic sliders in a matter of seconds without 
coding and image editing. 

More Info: http://wowslider.com/

How to create Wordpress slider with WOW Slider: http://wowslider.com/wordpress-jquery-slider.html

== Installation ==
Please see full tutorial here:
http://wowslider.com/wordpress-jquery-slider.html

1. Download WOWSlider plugin for Wordpress 
   http://wowslider.com/download/wowslider-wordpress-plugin.zip
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a slideshow in WOW Slider program and click "Publish" button, 
   select "Create Wordpress Slider" as a publishing method.
1. Add the created module (wowslider.zip) using WOW Slider plugin:
   WordPress Menu -> WOW Slider -> Add New Slider
1. You’ll see the following message: 
   "Slider added! To add it on the page use the shortcode: [wowslider id="3"]"
1. Add the shortcode inside the page where you want your slider to appear.  
1. Preview changes
